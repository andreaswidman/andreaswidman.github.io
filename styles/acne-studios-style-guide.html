06   Design Rules

Rule Value Rationale
Border Radius0px — no rounding anywhereSharp, architectural aesthetic
Box ShadowNoneFlat design language throughout
Base font size12px — unusually smallMaximises whitespace, editorial feel
Text transformuppercase for all UI labels / nav / buttonsBrand voice: precise, structured
Primary action colour#0018A8 (not black)Singular brand blue for all CTAs
PricesMonospace font (Helvetica Monospaced Pro)Tabular alignment, editorial clarity
Letter spacing0.3px universal defaultAir and legibility at small size
Line height20px base (1.67 ratio at 12px)Generous for body text at 12px
Grid gap0px on PLP gridImages bleed edge-to-edge
Image ratio3:4 (portrait) for all product imagesConsistent tile layout
Nav height60pxFixed header, sticky on scroll
Interactive link colour#3860BE (lighter blue)Fit predictor, anchors in body text
